Question-1: What are the different ways to select an element in the DOM?

Different ways to select an element in javascript DOM are getElementBy(),getElementByClassName(),getElementByTagName(),querySelector(),querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML returns HTML Tag, innerText returns Visible Text of an HTML Element, TextContent returns all Visible Text including Hidden Text.

Question-3: What is event delegation in the DOM?

Event delegation is a javascript technique, using this I can hanle events of childNodes and can add separate event litseners to each childs.

Question-4: What is event bubbling in the DOM?

Event bubbling is a DOM process where the event starts from the targeted element and propagates through it's parent and repeat untill it's reaches the document.

Question-5: How do you create, add, and remove elements using JavaScript?

We can create element using document.createElement() method, add element by using append(), appendChild() and innerHTML() methods, lastly we can remove or delete an element using remove() and removeChild() or innerHTML = "" methods.
